# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the MSE project, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, please email us at:

**opensource@godson.ai**

Include:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fixes (if applicable)

## Response Timeline

- **Acknowledgment:** Within 48 hours of report
- **Assessment:** Within 7 days
- **Fix:** Depending on severity, typically within 14-30 days
- **Disclosure:** Coordinated with the reporter after a fix is available
