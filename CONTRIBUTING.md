# Contributing to MSE

Thank you for your interest in contributing to the Moral Spectrometry Engine!

## Quick Links

- [Report a bug](https://github.com/godsons-ai/mse/issues/new?template=bug_report.md)
- [Request a feature](https://github.com/godsons-ai/mse/issues/new?template=feature_request.md)
- [Propose a new dilemma](https://github.com/godsons-ai/mse/issues/new?template=new_dilemma.md)
- [Discuss research questions](https://github.com/godsons-ai/mse/discussions)

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/mse.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b your-feature`
5. Make your changes
6. Run tests: `npm test`
7. Submit a pull request

## Areas Where Help Is Welcome

- New dilemma proposals (see [Dilemma Authoring Guide](./docs/DILEMMA_AUTHORING_GUIDE.md))
- Translations of dilemmas to additional languages
- LLM provider implementations
- Visualization components for `@godson/mse-react`
- Test coverage improvements
- Documentation enhancements

## Full Guidelines

For detailed contribution guidelines, coding standards, review process, and RFC procedures, see **[docs/CONTRIBUTING.md](./docs/CONTRIBUTING.md)**.

## Code of Conduct

Given that MSE deals with ethical reasoning, we expect contributors to model the behavior we evaluate. Be respectful, assume good faith, focus on substance, acknowledge uncertainty, and credit others' work.
